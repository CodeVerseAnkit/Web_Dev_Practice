DEFINATION OF HTML,CSS,JS

HTML
HTML (HyperText Markup Language) is the foundational coding language used to structure content on the web. It dictates how text, images, videos, and links are laid out in a web browser. It is not a programming language, but rather a markup language using tags to define elements.
CSS
CSS (Cascading Style Sheets) is a stylesheet language used to control the visual presentation, layout, and formatting of documents written in HTML or XML. While HTML builds the raw structure of a webpage, CSS is what makes it visually appealing by dictating colors, typography, spacing, and layouts.
JAVASCRIPT
JavaScript (JS) is a lightweight, interpreted programming language with first-class functions that is most famous as the scripting language for web pages. Alongside HTML (structure) and CSS (style), JavaScript serves as the core execution layer that brings interactivity, data calculation, and dynamic updates to modern web applications.
